Embora o livro tenha sido concebido para usar o Matlab, vou desenvolver os códigos usando o software livre equivalente [Octave](https://pt.wikipedia.org/wiki/GNU_Octave). 

## GNU Octave
é um software livre e ambiente de computação numérica de alto nível, semelhante ao MATLAB, usado para resolver problemas matemáticos, gráficos e de programação, sendo perfeito para engenharia, ciência e aprendizado de cálculo numérico, com uma linguagem simples e compatível com MATLAB, disponível para vários sistemas operacionais e com opção online. 

### Principais Características:

- Computação Numérica: Excelente para resolver problemas lineares, não-lineares, equações e experimentos numéricos.
- Compatibilidade com MATLAB: Código e funções muito similares ao MATLAB, facilitando a transição.
- Software Livre (Open Source): Gratuito, com código aberto, rodando em Windows, Linux, macOS e BSD.
- Interface: Oferece uma interface de linha de comando (CLI) e uma interface gráfica de usuário (GUI) com editor de scripts, histórico e visualização de gráficos.
- Extensível: Permite criar programas e estender funcionalidades com módulos em outras linguagens.
- Visualização: Ferramentas poderosas para gráficos 2D e 3D. 

### Para Que Serve (Uso):

- Estudantes e Pesquisadores: Estudar métodos numéricos, aprendizado de máquina, sistemas de controle.
- Engenharia e Ciência: Análise de dados, simulações, manipulação de vetores e matrizes.
- Programação: Escrever scripts para automação de tarefas matemáticas.
